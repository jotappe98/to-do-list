# To-Do List

Aplicação web de lista de tarefas com foco em organização, usabilidade e experiência do usuário.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Remover tarefas
- Marcar como concluída
- Filtro por status (todas, concluídas, pendentes)
- Busca de tarefas
- Estado vazio dinâmico (UX)
- Dark mode com toggle
- Persistência com LocalStorage
- Responsividade para dispositivos móveis

## Acesse o projeto

👉 https://jotappe98.github.io/to-do-list/

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome

## Destaques técnicos

- Manipulação de DOM
- Eventos (click, submit, keyup)
- Controle de estado no front-end
- Uso de LocalStorage para persistência
- Implementação de tema (light/dark)
- Estrutura organizada e reutilizável

## Responsividade

O layout foi adaptado para dispositivos móveis utilizando media queries, garantindo boa usabilidade em diferentes tamanhos de tela.
